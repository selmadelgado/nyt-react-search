# NYT-React Search
---------
<a name="overview"></a>
## Overview 
NYT-React is a React framework web app that allows users to search for articles via the New York Times API and to store and delete articles using MongoDB. 

<a name="technologies"></a>
## Technologies
 - Express.js 
 - Bootstrap
 - New York Times API
 - React 4 
 - Mongoose 
 - MongoDB
 	- Local database
 - mLab 
 	- Deployed Heroku database
